boxMap

Download this project as a .zip file Download this project as a tar.gz file

boxMap aims to provide a tool to accurately detect and localize middleboxes in the Internet. It is a client server application using probe-feedback mechanism to probe different Internet paths and post the packet impairment results to a master node. A first implementation will be deployed on Planetlab and Ark testbeds.
